This archive contains some examples for fax cover pages useable with YajHFC.
You may freely use and modify them in order to create your own cover pages.

- Coverpage example.html: An example for a simple HTML cover page including a logo
- coverpage.html        : The YajHFC default cover page beginning with version 0.4.2
- coverpage_de.html     : The YajHFC default cover page beginning with version 0.4.2, German translation
- faxcover.ps           : The YajHFC default cover page for versions below 0.4.2
- faxcover_de.ps        : The YajHFC default cover page for versions below 0.4.2, German translation
